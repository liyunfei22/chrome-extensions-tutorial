安装 Chrome 拓展 #

大多数 Chrome 用户从 Chrome 应用商店获取扩展程序,也可以在本地机器上安装

本地机器上安装 #

  1. 在浏览器中导航到 chrome://extensions。还可以通过单击多功能框右上角的 Chrome 菜单、将鼠标悬停在更多工具上并选择扩展来访问此页面

  1. 选中开发者模式，单击加载已解压的扩展程序
